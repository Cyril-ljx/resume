# 个人简历网页 - 设计方案

## 样例分析

根据提供的样例图片，该简历网页具有以下特点：

### 视觉风格
- **配色方案**：深紫色背景 + 青蓝色/浅蓝色文字和装饰元素
- **设计风格**：现代科技感、极简主义
- **背景**：深紫色渐变背景，配有散落的装饰点（圆形）
- **文字颜色**：白色标题 + 青蓝色副标题 + 浅灰色正文

### 页面结构（全屏滚动）
1. **首页（Hero）**
   - 大号姓名/名字（如"黎子聪"）
   - 职位标题（如"AI大模型应用工程师"）
   - 个人slogan
   - 技术栈标签（圆形边框，青蓝色文字）
   - 装饰圆点

2. **关于我（About）**
   - 标题"关于我"
   - 圆形头像（青蓝色边框）
   - 教育背景和个人介绍文本
   - 黑色背景

3. **核心技能（Skills）**
   - 标题"核心技能"
   - 技能列表，每项包含：
     - 图标（如代码符号、大脑、数据库、芯片等）
     - 技能名称
     - 进度条（渐变色：青蓝→紫色）
     - 百分比数字

4. **科研与成果（Research）**
   - 标题"科研与成果"
   - 学术论文卡片（可点击跳转）
   - 卡片包含：
     - 论文标题（青蓝色，可点击）
     - 期刊名称
     - 作者身份
     - 简要描述
     - 标签（如"SCI四区"）

5. **项目历程（Projects）**
   - 时间范围
   - 项目名称
   - 项目描述
   - 技术标签

6. **实践经历（Experience）**
   - 时间范围
   - 公司/机构名称
   - 职位
   - 工作描述

7. **联系方式（Contact）**
   - 标题"联系方式"
   - 各种联系方式（邮箱、电话、社交媒体等）

## 设计决策

### 配色方案
- **主背景**：深紫色（#1a1a2e 或类似）
- **主色调**：青蓝色/浅蓝色（#00d4ff 或类似）
- **强调色**：紫色（#9d4edd 或类似）
- **文字**：白色（#ffffff）、浅灰色（#b0b0b0）
- **边框/装饰**：青蓝色

### 字体选择
- **标题**：粗体无衬线字体（如 Poppins Bold、Roboto Bold）
- **正文**：中等无衬线字体（如 Poppins Regular、Roboto Regular）
- **中文**：系统字体或 ZCOOL 系列

### 动画效果
- 全屏滚动时的淡入淡出效果
- 卡片悬停时的阴影和缩放效果
- 进度条的动画加载
- 装饰圆点的浮动动画

### 响应式设计
- 移动端：单列布局
- 平板：两列布局
- 桌面：优化的多列布局

## 技术实现

- **框架**：React 19 + Tailwind CSS 4
- **路由**：Wouter（客户端路由）
- **滚动效果**：Framer Motion 或原生 CSS
- **图标**：Lucide React
- **组件库**：shadcn/ui

## 内容占位符

所有内容使用示例数据，用户可后续替换：
- 姓名、职位、slogan
- 头像图片
- 教育背景、技能、项目、经历
- 科研成果链接
- 联系方式
